# Автоматизация поиска информации о Земле на Wikipedia

Этот скрипт автоматизирует поиск информации о содержании кислорода в атмосфере Земли на русской Wikipedia.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что у вас установлен Chrome браузер

3. Selenium автоматически загрузит ChromeDriver через менеджер драйверов

## Запуск

```bash
python wikipedia_automation.py
```

## Что делает скрипт

1. Открывает https://ru.wikipedia.org
2. Вводит "Земля" в поле поиска
3. Переходит на статью "Земля"
4. Ищет информацию о содержании кислорода в атмосфере (20,95%)
5. Выделяет найденную информацию
6. Делает скриншот и сохраняет его как `wikipedia_earth_oxygen.png`

## Запуск Playwright теста

Для запуска теста с использованием Playwright Test:

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите браузеры для Playwright:
```bash
playwright install
```

3. Запустите тест:
```bash
pytest test_wikipedia_earth.py
```

## Что проверяет Playwright тест

Тест `test_wikipedia_earth.py` проверяет:

1. Открывает https://ru.wikipedia.org
2. Вводит "Земля" в строку поиска
3. Переходит на статью "Земля"
4. Проверяет, что в статье НЕТ текста "50,00" (неправильное значение содержания кислорода)

Тест использует `page.locator`, `expect` и Playwright Test.
